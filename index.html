    Malbian Linux - Documentation #

    This is the documentation page of Malbian Linux, a GNU/Linux distribution for Malware Analysis and Reverse Engineering that Launched January 11th 2025.

    Why Malbian? #

    Malbian is a distribution used for analyzing and triaging malware samples. It can be run or installed in any Hypervisor due to how it’s images are created and distributed. It’s based on Debian and it offers a set of custom scripts that would run on a Tmux session to provide information about the analysis environment.

    Malbian offers a very fast and lightweight ISO that runs with Suckless Software, or a very user friendly alternative with an XFCE Desktop Environment.

    Also, Malbian is in an Alpha state, meaning that we are open to changes and/or modifications from the community to best suit our needs. Anyone can participate, suggest or ask for tools to be added into Malbian’s images.
