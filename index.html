    Malbian Linux - Documentation #

    This is the documentation page of Malbian Linux, a GNU/Linux distribution for analyzing and triaging malware samples that Launched in early January 2025.

    Why Malbian? #

    Malbian is a light-weight and fast distribution that can be installed in any Hypervisor due to how it’s images are created and distributed. It’s based on Debian and it offers a set of custom scripts that would run on a Tmux session to provide information about the analysis environment.

    Also, Malbian is in an Alpha state, meaning that we are open to changes and/or modifications from the community to best suit our needs. Anyone can actually participate, suggest or ask for tools to be added into Malbian’s images.
